# まとめてアラーム (Compact Alarm)

まとめてアラームは、朝の起床時に複数回アラームをセットして起きたい方に最適なAndroidアプリです。
一つのカードに複数のアラームを設定し、それらを一つのスイッチで簡単に管理することができます。

## 特徴

- **簡単な管理**: 一つのカードに複数のアラームをセットして管理できます
- **カスタマイズ可能**: 例えば、朝6:00に起きる場合、5:50、5:55、6:00といった時間を一つのカードに追加可能
- **一括オンオフ**: 設定した全アラームを一つのスイッチで簡単にオン・オフ
- **曜日指定**: アラームを特定の曜日のみ鳴らすように設定可能
- **モダンなUI**: Material Design 3に準拠したダークテーマ対応のUI

## 技術スタック

- **言語**: Kotlin 1.9.22
- **UI**: Jetpack Compose + Material 3
- **アーキテクチャ**: MVVM + Repository パターン
- **DI**: Hilt
- **データ永続化**: DataStore Preferences
- **最小SDK**: 26 (Android 8.0)
- **ターゲットSDK**: 34 (Android 14)

## ビルド要件

- Android Studio Hedgehog (2023.1.1) 以降
- JDK 17
- Gradle 8.5

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/yourusername/compact-alarm.git
```

2. Android Studioでプロジェクトを開く

3. Gradleの同期を実行

4. デバイスまたはエミュレータでアプリを実行

## アプリの使い方

1. 画面下部の「+」ボタンをタップして、アラーム時刻を設定
2. 「セット」をタップしてアラームを保存
3. ホーム画面に表示されたアラームカードの「時間を追加」をタップで同じグループに追加のアラームを設定
4. 曜日ボタンをタップして、アラームを鳴らす曜日を選択
5. スイッチでアラームのオン/オフを切り替え
6. 左にスワイプでアラームを削除

## 必要な権限

- **正確なアラーム**: 正確な時刻にアラームを鳴らすために必要
- **通知**: アラーム通知を表示するために必要
- **起動時の実行**: 端末再起動後にアラームを復元するために必要
